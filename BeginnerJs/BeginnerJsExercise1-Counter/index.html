Fancy Counter

0

Code challenge: Build a fancy counter application using HTML, CSS, and JavaScript. Implement the code in Javascript, creating a script.js file already linked in the HTML. use functions to organize your code better: increment button will increment the counter__value by 1 when clicked, decrement button will decrement the counter__value by 1 when clicked, reset button will reset the counter__value to 0 when clicked. if counter__value will reach any number above 0, the counter__value text color will change to green, if counter__value value will be above 5, the counter__value text color will change to red, and counter__title text will become : "Limit! Buy Pro for >5". Increase and decrease will be set to disabled. if counter__value will reach any number below 0, the counter__value text color will change to blue. clicking decrease button when counter__value is 0 or less will have no effect (it will remain to zero). when reset button is clicked, counter__value text color will return to default color, and counter__title text will return to "Fancy Counter". Increase and decrease will be set to enabled.
